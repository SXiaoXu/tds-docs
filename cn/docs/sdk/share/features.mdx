## 功能概述
分享功能支持游戏调起 TapTap 的内容编辑器快捷发布图文内容。
支持预设标题、正文、话题、论坛标签信息，玩家编辑时将默认带上预设选项。便于更好的聚类、管理分享内容。

![](https://img.tapimg.com/market/images/f53fa5245bc3eb20a8f1a21d911ebc46.png)

## 服务优势
- 便捷分享，帮助游戏内容在 TapTap 传播
    - 一键拉起 TapTap 内容编辑器，快捷完成分享。
- 预设分享信息，内容征集更简单
    - 支持预设多个内容发布选项，帮助玩家圈定分享范围，便于运营管理。
- 获得「资源置换」流量扶持
    - 用户通过分享进入 TapTap ，将产生拉新拉活价值奖励，可用于换取游戏在 TapTap 的曝光流量。

:::info
流量扶持奖励需在 [TapTap REP 平台](https://rep.taptap.cn/) 领取使用。

:::

## 使用场景示例
- 战力截图分享活动举例
    - 游戏中战力界面截图，选择分享至 TapTap ，先获得一次游戏内分享奖励。
    - 分享图文预设「最强战力评选」话题，「截图分享」论坛标签，「我的战力最强，不服来战！」标题。
    - 同时在 TapTap 内发起「秀出战力」内容征集社区活动，从分享用户中选取获奖用户公示发放奖励。

![](https://img.tapimg.com/market/images/12c5199e9385c25f579d289a8324929f.png)

|---------------|-----------------------|-----------------|
|论坛标签|将发布在游戏论坛的该标签下。通过 TapTap 论坛标签页的网址末尾网址数字，可获取该 ID。![](https://img.tapimg.com/market/images/d7d71d0c3ff67adc600749f159c9c1a7.png
)|![](https://img.tapimg.com/market/images/7b4bee1af10c8f125980ab1d481db24c.png)|
|话题|将带上该话题，该话题列表中将包含该内容。话题 ID 请通过工单选择分类「TDS 游戏服务 - 分享 - 查询话题 ID」，提供具体话题名称进行人工查询。|![](https://img.tapimg.com/market/images/25348a285d931ef9c2dde7c08de1736e.png)|
|标题与正文|将预填充所设置的标题与正文。|![](https://img.tapimg.com/market/images/b63db9640f4335a6cd7938a9e9be968e.png)|

## 注意事项
- 当用户没有 TapTap 客户端而导致分享失败时，默认将打开网页版的游戏详情页。
- 你可以自定义该场景跳转的链接。推荐使用 REP 生成的游戏详情页效果链接，详情可前往 [REP 资源置换平台](https://rep.taptap.cn/) 查看。
