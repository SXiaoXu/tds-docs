<h2> 4.5.0 </h2>

更新时间 2024-11-18

<h4> 功能 </h4>

- [Android] 新增 [ 内购服务 ] ：提供整合商品管理、支付渠道、订单管理的便捷内购解决方案


<h4> 优化 </h4>

- 合规认证：当 Startup 接口异步处理未结束时，如果重复调用该接口，SDK 将直接返回
- 删除冗余 Protobuf 依赖
- [iOS] 内嵌动态：添加隐私说明文件，删除在 framework 中冗余的资源文件
- [PC] TapTap 登录：更新内部网络库及登录页面扫码引导图

<h4> 修复 </h4>

- [Android] 修复 OAID 参数设置无效问题
- [Android] 数据分析：修复调用仅包含用户 ID 的 SetUserID 接口时的异常问题

<h2> 4.4.3 </h2>

更新时间 2024-10-30

<h4> 优化 </h4>

- [PC&iOS] 优化内部处理，避免与阿里云 SLS SDK 、字节巨量 SDK 冲突

<h4> 修复 </h4>

- [PC] 修复使用 Net Framework 打包时编译错误
- [iOS] TapTap 分享：修复最低支持版本与其他模块不一致问题
- [iOS] 数据分析：修复多次上报设备更新事件的问题
- [Android] TapTap 分享：修复单独集成时运行崩溃问题
- [Android] 数据分析：修复单独集成时数据上报失败的问题
- [Android] 正版验证：修复同时安装国内和海外 TapTap 客户端时调用检查购买接口时未响应问题
